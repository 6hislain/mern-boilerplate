# MERN boilerplate

A minimal MERN stack boilerplate

## Requirements

- node 20
- mongodb
- code editor
- browser

## Installation

- git clone https://github.com/6hislain/mern-boilerplate
- cd mern-boilerplate/client
- npm install
- cd ../server
- npm install
- cd ..
- npm i -g concurrently
- concurrently "npm run dev --prefix client" "npm run dev --prefix server"
